# soutez.teleinformatika.eu
Webová prezentace krajské soutěže zaměřené na digitální tvorbu (Blender, Krita, Illustrator). Obsahuje informace o pravidlech, programu, hodnocení a organizaci soutěže.
# 🏆 Soutěž tvůrčích nástrojů

Webová prezentace krajské soutěže pořádané Střední školou teleinformatiky v Ostravě, zaměřené na digitální tvorbu a kreativní nástroje.

## 📌 O projektu

Tento web slouží jako centrální místo pro účastníky soutěže. Obsahuje důležité informace o průběhu akce, pravidlech, kategoriích a hodnocení.

Soutěž je zaměřena na:
- 🎨 Grafiku (Adobe Illustrator)
- 🖌️ Digitální malbu (Krita)
- 🧊 3D tvorbu (Blender)
